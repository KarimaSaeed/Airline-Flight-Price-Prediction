# Airline-Flight-Price-Prediction
This project builds an end-to-end machine learning regression model to predict airline ticket prices. The workflow includes exploratory data analysis (EDA), data preprocessing (handling missing values, outliers, encoding, and scaling), and model training using Scikit-learn. Model performance was evaluated using RMSE and R².
The goal is to analyze historical flight data and identify the key factors that influence ticket pricing, enabling better understanding of pricing patterns.

---

## 🔍 Exploratory Data Analysis (EDA)
- Performed univariate and multivariate analysis
- Identified relationships between features such as airline, duration, stops, and price
- Visualized trends and patterns using Matplotlib and Seaborn

---

## 🧹 Data Preprocessing
- Handled missing values and inconsistent data
- Detected and treated outliers
- Encoded categorical variables (e.g., airline, source, destination)
- Applied feature engineering and scaling techniques

---

## 🤖 Model Development
- Built regression models using **Scikit-learn**
- Trained models to predict continuous flight prices
- Evaluated model performance using regression metrics

---

## 📈 Results
- Evaluated model performance using:
  - RMSE (Root Mean Squared Error)
  - R² Score
- Generated insights on how different features impact flight pricing

---

## 💡 Key Insights
- Flight duration and number of stops significantly affect ticket prices  
- Airline and route (source/destination) play a major role in pricing  
- Feature engineering improves model performance  

---

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
